To install this addon downlaod the files and copy them to your wow/Interface/AddOns folder.
To use it, simply type /ExportEPGP in to your in game chat
This opets up a little window with soem text in it. Click on the text, press CTRL + A then CTRL + C.
Alt tab out of yoru game and paste this data on the apropriate page on our webpage.
The text is pre formated and ready to be used as is.
There is currently no way of closing the windwo smoothy,
So to close the window click your chat and do /reload.
